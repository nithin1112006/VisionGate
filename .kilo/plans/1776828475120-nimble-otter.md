# VPN Blocking Implementation Plan

## Overview
Implement VPN detection and blocking for the StaffSync attendance system across all platforms (Android, iOS, Web, Windows).

## Technical Architecture

### 1. Client-side Detection (Flutter)
**File**: `siet_sync/lib/utils/vpn_check.dart`
- **Android**: Use `connectivity_plus` to detect `ConnectivityResult.vpn`
- **iOS**: Check for `utun*` network interfaces (used by all iOS VPNs)
- **Windows**: Check interface names for VPN indicators
- **Web**: Server-side detection only (client-side WebRTC leak detection optional)

### 2. Settings Integration
**Modify**: `siet_sync/lib/utils/wifi_check.dart`
- Add `enforceVpnBlocking` boolean to `AppSettings` class
- Update settings parsing regex
- Add to `/settings/allow_any_network` endpoint response

### 3. Backend Integration
**Modify**: `backend/main.py`
- Add `enforce_vpn_blocking` to `_app_settings`
- Update settings endpoints
- Add optional IP reputation checking middleware

### 4. Validation Flow Integration
**Modify existing validation calls**:
- `face_verification_service.dart:185` - Add VPN check before attendance
- `face_registration_widget.dart:628` - Add VPN check before registration
- Extend `validateCollegeWifi()` to include VPN validation

### 5. UI/UX
- Add blocking alert dialog with message: "VPN detected. Please turn off your VPN to continue."
- Integrate VPN status into existing network status displays
- Follow existing error pattern for consistency

## Exact Code Changes

### 1. siet_sync/lib/utils/wifi_check.dart
Add to AppSettings:
```dart
static bool enforceVpnBlocking = true;
```

Add parsing in parseJson():
```dart
final vpnMatch = RegExp(r'"enforce_vpn_blocking"\s*:\s*(true|false)').firstMatch(body);
if (vpnMatch != null) {
  result['enforce_vpn_blocking'] = vpnMatch.group(1) == 'true';
}
```

Update in loadSettings():
```dart
enforceVpnBlocking = data['enforce_vpn_blocking'] ?? true;
```

### 2. backend/main.py
Add to _app_settings:
```python
"enforce_vpn_blocking": True,
```

Update settings endpoints:
```python
"enforce_vpn_blocking": _app_settings.get("enforce_vpn_blocking", True),
```

### 3. Extend validation
Add to validateCollegeWifi():
```dart
final vpnError = await VpnChecker.validateVpnStatus();
if (vpnError != null) return vpnError;
```
